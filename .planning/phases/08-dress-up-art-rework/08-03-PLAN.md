    Per CONTEXT.md "Fresh start: delete old generated PNGs/SVGs, regenerate everything from scratch":

    1. Delete old generated assets:
       ```bash
       rm -rf assets/generated/png/dressup/
       rm -rf assets/generated/svg/dressup/
       rm -f frontend/assets/svg/dressup/*.svg
       rm -f frontend/assets/svg/mermaid.svg
       ```

    2. Run the generation pipeline (requires OPENAI_API_KEY env var):
       ```bash
       cd /Users/chriskaschner/Documents/GitHub/mermaids
       uv run python -c "from mermaids.pipeline.edit import generate_dressup_variants; generate_dressup_variants()"
       ```
       This generates 1 base mermaid PNG + 12 part variant PNGs via gpt-image-1 edit API.

    3. Trace all PNGs to SVG:
       ```bash
       uv run python -c "
       from pathlib import Path
       from mermaids.pipeline.trace import trace_to_svg
       from mermaids.pipeline.config import GENERATED_PNG_DIR, GENERATED_SVG_DIR

       # Trace base body
       base_png = GENERATED_PNG_DIR / 'dressup' / 'base' / 'mermaid-base.png'
       base_svg_dir = GENERATED_SVG_DIR / 'dressup'
       base_svg_dir.mkdir(parents=True, exist_ok=True)
       trace_to_svg(base_png, base_svg_dir / 'body.svg')

       # Trace all part variants
       parts_dir = GENERATED_PNG_DIR / 'dressup' / 'parts'
       for png in sorted(parts_dir.glob('*.png')):
           trace_to_svg(png, base_svg_dir / f'{png.stem}.svg')
       print('Tracing complete.')
       "
       ```

    4. Assemble multi-layer mermaid.svg:
       ```bash
       uv run python -c "
       from mermaids.pipeline.assemble import assemble_mermaid_svg
       from mermaids.pipeline.config import GENERATED_SVG_DIR, FRONTEND_SVG_DIR

       traced_dir = GENERATED_SVG_DIR / 'dressup'
       base_svg = traced_dir / 'body.svg'
       output = FRONTEND_SVG_DIR / 'mermaid.svg'
       assemble_mermaid_svg(traced_dir, output, base_traced_svg=base_svg)
       print('Assembly complete.')
       "
       ```

    5. Copy individual part SVGs to frontend for preview thumbnails:
       ```bash
       uv run python -c "
       from mermaids.pipeline.assemble import copy_dressup_parts_to_frontend
       results = copy_dressup_parts_to_frontend()
       print(f'Copied {len(results)} preview SVGs.')
       "
       ```

    6. Verify file counts:
       - frontend/assets/svg/mermaid.svg exists
       - frontend/assets/svg/dressup/ has 12 SVG files (tail-1..3, hair-1..3, eye-1..3, acc-1..3)
